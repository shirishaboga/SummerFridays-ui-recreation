# Cosmetic Website UI Recreation

## Overview
This project is a UI recreation of the "Summer Fridays" cosmetic website. The goal was to design a clean, modern, and visually appealing interface inspired by real-world website layouts.

## Description
In this project, I focused on recreating the interface of the "Summer Fridays" cosmetic website with attention to layout, spacing, and overall aesthetics. The design includes basic scrolling interaction to give a more realistic user experience.

## Features
- Clean and minimal UI design  
- SummerFridays visual style  
- Basic scrolling interaction  
- Structured layout for better user experience
